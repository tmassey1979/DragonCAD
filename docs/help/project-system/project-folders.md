# Project Folders

DragonCAD project folders keep workspace state, design files, generated outputs, and supporting assets together.

Use this page when validating what belongs in a project directory or when moving a design between workstations.

## Stored files

- `dragoncad.project.json` contains the project manifest.
- Workspace shell state preserves editor placement and routing context.
- Generated fabrication and sourcing artifacts remain local until explicitly exported.

Return to [Workspace basics](../getting-started/workspace.md) when orienting a project from a fresh checkout.
