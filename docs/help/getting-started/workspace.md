# Workspace Basics

Open a DragonCAD workspace to inspect project state, switch between schematic and board work, and review application status.

## Daily workflow

- Open or create a local project folder before starting design work.
- Use the workspace tabs to move between components, schematic capture, PCB layout, fabrication, and help.
- Keep an eye on the footer status text before saving or exporting artifacts.

Project folder behavior is covered in [Project folders](../project-system/project-folders.md).

```text
File > Save
File > Open Project Folder
```
